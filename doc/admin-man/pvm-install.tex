%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\subsection{\label{sec:Install-PVM-Condor}
Installing PVM Support in Condor} 
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\index{contrib module!PVM}
\index{PVM contrib module}
\index{installation!PVM contrib module}
To install support for PVM in Condor, download the file archive from
\Url{http://www.cs.wisc.edu/condor/downloads} and follow the directions
found the \File{INSTALL} file contained in the archive.
\Note The PVM contrib module version must agree with your installed
Condor version.
