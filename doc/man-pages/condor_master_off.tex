\begin{ManPage}{\label{man-condor-master-off}\Condor{master\_off}}{1}
{Shutdown Condor and the \Condor{master}}
\Synopsis \SynProg{\Condor{master\_off}}
\oOpt{-help}
\oOpt{-version}
\oOpt{hostname ...}

\index{Condor commands!condor\_master\_off}
\index{condor\_master\_off command}

\Description 

\Condor{master\_off} no longer exists.

\GenRem
\Condor{master\_off} no longer exists as a Condor command.
Instead, use
\begin{verbatim}
        condor_off -master
\end{verbatim}
to accomplish this task.

\SeeAlso
See the \Condor{off} manual page.
